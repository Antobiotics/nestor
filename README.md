# Nestor


Prerequisite 
--------------------------------------------------------

Install [RUST](https://www.rust-lang.org/) from https://www.rust-lang.org/downloads.html

Reference Commands with [Cargo](http://doc.crates.io/)
--------------------------------------------------------

#### Project Clean

```bash
cargo clean
```

#### Build

```bash
cargo build
```

#### Unit Test

```bash
cargo test
```

#### Run

```bash
cargo run
```
